This exercise will test your ability to implement responsive CSS design. You're free to search for any
reference you might need online, but the actual assignment itself should be done primarily by modifying
css/main.css, and as little of the html as you can.

####################################################################################################
## ENVIRONMENT
####################################################################################################
You have available to you Bootstrap 3.1, and a skeleton of a fluid grid layout (currently 2-column).
You will be required to modify this grid according to the instructions below. The breakpoints we will
use are as follows:
0 - 480px : mobile view
481px - 767px : phablet view
768px - 991px : tablet view
992px - larger: desktop view

You can use the Bootstrap classes where needed or write your own CSS, as you prefer. The grid must
always be fluid and should not use fixed widths / heights.


####################################################################################################
## TASK #1
####################################################################################################
Modify the whole grid (.grid) so that it can only stretch up to 1280px, maximum


####################################################################################################
## TASK #2
####################################################################################################
Align the text in each grid item to be center-aligned horizontally and vertically


####################################################################################################
## TASK #3
####################################################################################################
Refer to the wireframes provided, and modify the grid to match them for each given breakpoint. The 
images correspond to the previously mentioned viewports:

 - original.png: This is the starting point of this test
 - 480.png: 480px and smaller (mobile)
 - 481.png: 481px - 767px (phablet)
 - 768.png: 768px - 991px (tablet)
 - 992.png: 992px and larger (desktop)
