There are 2 separate tasks in this exercise, under /backend and /frontend, respectively. You will need 
a PHP5 / MySQL webserver. 

Before beginning the backend task, you will need to set up an empty database and import /backend/db.sql.
You will also need to update the DB credentials in /backend/functions.php, under $CONFIG. If properly
set up, you should be able to load /backend/index.php and see a list of books.

Further instructions are under /backend/README.txt and /frontend/README.txt

Upon completion of the tasks, please export your DB, and send it together with your code.
